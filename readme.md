# Basic gRPC Service written in Python

## Prerequisites
- Python 3.7 or higher
- pip

## Installation

```shell
~ mkcert -install
~ mkcert -cert-file ./certs/local.crt -key-file ./certs/local.key localhost 127.0.0.1 0.0.0.0 ::1
```

```shell
~ python -m pip install -e .
```
